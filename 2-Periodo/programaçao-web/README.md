# 💻 Programação Web - Trabalhos e Atividades

Bem-vindo ao meu repositório de projetos e atividades da disciplina de Programação Web! Aqui você encontrará uma coleção de trabalhos desenvolvidos ao longo do curso, explorando diversas tecnologias e conceitos fundamentais do desenvolvimento web, além de materiais disponibilizados pelo professor ao longo do curso.

## 📚 Sobre o Repositório

Este repositório contém projetos práticos e exercícios que cobrem desde os fundamentos do desenvolvimento web até conceitos mais avançados. Cada projeto representa uma etapa do meu aprendizado e evolução como desenvolvedor.

Grande parte dos arquivos são um acervo de arquivos disponibilizados pelo professor da diciplina ao longo do semestre.

**Tecnologias:** HTML, CSS, JavaScript, React, Node.js, MongoDB

## 📖 Aprendizados

Durante o desenvolvimento destes projetos, tive a oportunidade de aprender e praticar:

- ✅ Desenvolvimento de interfaces responsivas
- ✅ Manipulação do DOM com JavaScript
- ✅ Integração com APIs REST
- ✅ Conceitos de design e UX
- ✅ Versionamento de código com Git
- ✅ Boas práticas de desenvolvimento web
